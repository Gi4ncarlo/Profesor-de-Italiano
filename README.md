# El Rincón de Luci 🇮🇹

Plataforma privada de aprendizaje de italiano para **Giorgio** y **Luciana**.

## Design System

Este proyecto utiliza un sistema de diseño personalizado inspirado en la calidez de una trattoria o un caffè romano.

### Estética
- **Colores:** Crema, Terracota, Espresso, Oliva, Dorado, Arena.
- **Tipografía:** Playfair Display (Títulos) y Crimson Text (Cuerpo).
- **Textura:** Papel con gránulo ligero (light noise).

### Archivos clave:
- `design-system.css`: Contiene todas las variables CSS y estilos de componentes reutilizables.
- `design-guide.html`: Una guía visual dinámica de los elementos del sistema. Abrir en el navegador para previsualizar.

## Estructura
- `/index.html`: Pantalla de entrada y selección de perfil.
